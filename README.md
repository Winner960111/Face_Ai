# FaceAI

一款优秀的人脸检测、人脸识别、视频识别的智能AI项目。

>开发环境：Windows 10（x64） + Python 3
>其他库版本：OpenCV 3.4.1、Dlib 19.8.1、Tesseract OCR 4.0.0-beta.1

## 技术实现方案 ##

下面列举一下，相应模块使用到技术.

人脸识别：OpenCV / Dlib

人脸检测：face_recognition

视频识别：OpenCV + face_recognition

Tesseract Ocr文字识别
